Python Developer

  • JavaScript / TypeScript / HTML / CSS
  • Amazon Connect
  • ElasticSearch Logstash Kibana and some Splunk
  • CloudWatch,
  • IAM, AWS Config Rules

We value technologies and individuals who can rapidly bring value to our customers.
